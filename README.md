# Mediawiki Category Readability

## Description

Category Readability ranks the articles in a category on a mediawiki site by their readability.
This is a toy project created as a test.

## Requirements

* PHP >= 5.3.0
* Composer
* The mediawiki site being ranked must have the [TextExtracts](https://www.mediawiki.org/wiki/Extension:TextExtracts) extension installed

## Installation

* Clone or unzip the source code in a directory accessible to your web server
* Run `composer install` in the source directory to download dependencies and set up the autoloader
